## Packages
react-qr-code | Generating QR codes for student orders
framer-motion | Smooth animations for page transitions and interactions
lucide-react | Beautiful icons for the UI

## Notes
- Images should be from Unsplash with descriptive keywords
- Auth is handled via Replit Auth (/api/login, /api/logout)
- Cart state is managed client-side using React Context
- Order flow: Student -> Cart -> Pre-order -> QR Code -> Shopkeeper Scan -> Confirm/Print
